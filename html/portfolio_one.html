Predicting Movie Ratings

Sentiment Analysis

The intent of this natural language processing problem was to predict the ratings of a movie-goer based on his/her review. In order to yield the most accurate results, I explored combinations of various pre-processing techniques on a training data set. Approaches include lemmatizing, stemming, removing stopwords, observing word frequency and inverse document frequency. With the feature matrix engineered, I trained models using logistic regression, a support vector machine of a linear kernel, and a naive bayes and used the most accurate models on a test set.

Shipping Yard

Nonlinear Search Planner

Plan out a strategy for moving containers from a ship to specific locations in a shipping yard. Because a linear planner is inefficient in determining a solution for (this type of problem category), I used a nonlinear partial order planner. Initially I began with empty partial plans. I then continuously added causal links, which connect states, until a partial plan has met all conditions and is considered complete.

An Unpredictable World

Markov Decision Process

Direct an agent to a goal state in an unpredictable environment. Due to the environment’s indeterminateness, an optimal policy was determined for the agent via value iteration. The expected utility for each action at any given position is calculated. The action with the greatest expected utility is used as the optimal policy for that position and is used to update the policy for that position (along with value at position). Policies were are updated until convergence. Monte Carlo simulations were run to verify the model’s acceptability.
