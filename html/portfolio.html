Predicting Movie Ratings

Sentiment Analysis

The intent of this natural language processing problem was to predict the ratings of a movie-goer based on his/her review. In order to yield the most accurate results, I explored combinations of various pre-processing techniques on a training data set. Approaches include lemmatizing, stemming, removing stopwords, observing word frequency and inverse document frequency. With the feature matrix engineered, I trained models using logistic regression, a support vector machine of a linear kernel, and a naive bayes and used the most accurate models on a test set.

Shipping Yard

Nonlinear Search Planner

Plan out a strategy for moving containers from a ship to specific locations in a shipping yard. Because a linear planner is inefficient in determining a solution for (this type of problem category), I used a nonlinear partial order planner. Initially I began with empty partial plans. I then continuously added causal links, which connect states, until a partial plan has met all conditions and is considered complete.

An Unpredictable World

Markov Decision Process

Direct an agent to a goal state in an unpredictable environment. Due to the environment’s indeterminateness, an optimal policy was determined for the agent via value iteration. The expected utility for each action at any given position is calculated. The action with the greatest expected utility is used as the optimal policy for that position and is used to update the policy for that position (along with value at position). Policies were are updated until convergence. Monte Carlo simulations were run to verify the model’s acceptability.

The Traveling Salesman

A* Search Algorithm

The classic traveling salesman problem is an NP-hard problem that requires(?) the shortest route to connect a set of points. I used the A* search algorithm because it is an admissible heuristic. Several heuristics were explored to help efficiently determine the optimal path. For estimating the cost of the remaining (sparse graph) path, Kruskal’s algorithm was found most useful. A mininum spanning tree was created and the heuristic allowed for pruning of excessive length paths.

Bayesian Inference

Given a library of articles, I was able to attribute an author to anonymous works. To ignore the bias of the article content and focus on a writer’s patterns, I used stopword frequency as a measurement. Stop word frequency was a much better predictor of authorship over term frequency.

Finite-state Machine

Assembly Level Programming

A finite-state machine was implemented to carry out a computer's assembly instructions. I implemented procedure calls in an assembly language similar to ARM and designed an optimal finite-state machine. Instructions were fetched from memory, decoded and executed until the program reaches a halt.

Pipelined Datapath

Parallel Instruction Clocking Scheme

By implementing a pipelined datapath I reduced the execution time for data processing that a single cycle and cycle multicycle data path could not. The five stage pipeline was complete with data forwarding and branch prediction, to prevent data hazards and control flow hazards. By allowing for the register files and memory writes to be present for the entire cycle, forwarding can be used to update instruction dependencies.
